# 📊 Projeto: Dashboard Interativo de Dados

Este repositório contém o projeto final desenvolvido para o Bootcamp de Engenharia de Dados. O objetivo principal foi estruturar os dados e criar uma planilha visual interativa (Dashboard) que permite a análise de indicadores de forma dinâmica.

---

## 🛠️ Recursos Utilizados no Excel/WPS Office
* **Tabela Dinâmica:** Para consolidação e tratamento dos dados matemáticos.
* **Gráfico Dinâmico:** Para visualização gráfica dos resultados.
* **Segmentação de Dados (Slicers):** Cartões clicáveis que funcionam como filtros dinâmicos para o painel.

---

## 🚀 Como Visualizar e Interagir com a Planilha Visual

Como o GitHub não renderiza os gráficos e botões interativos do Excel diretamente no navegador, siga os passos abaixo para testar o projeto:

### Passo 1: Baixar o Arquivo
1. Na página inicial deste repositório, clique no arquivo **`.xlsx`** (com o nome do seu arquivo).
2. No canto direito da tela, clique no botão **Download** (ícone de uma seta para baixo) para salvar o arquivo no seu computador.

### Passo 2: Abrir no Software de Planilhas
1. Abra o arquivo baixado utilizando o **Microsoft Excel** ou o **WPS Office**.
2. Caso apareça um aviso no topo da tela, clique em **"Habilitar Edição"** para que os recursos interativos funcionem.

### Passo 3: Interagir com o Dashboard
1. Vá para a aba chamada **`Dashboard`**.
2. Clique nos **Cartões Clicáveis (Botões de Segmentação)** no topo ou na lateral da tela.
3. Note que ao clicar em um botão, o gráfico e os valores em Reais (`R$`) se atualizam automaticamente na hora.
